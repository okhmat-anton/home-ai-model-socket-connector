# Home AI Model Socket Connector — Instruction for AI Agent

This service is a bridge between you (an AI model) and the outside world.
It has two modes:

1. INCOMING — a user sends you a request through the server.
2. OUTGOING — you can access the internet through the server's HTTPS proxy.

Server: {server_ip}:{port}
Proxy: {server_ip}:{proxy_port}
Connected models: {connected_models}
Default model (first connected): {default_model}

---

## 1. Send a Request to the AI Model (INCOMING MODE)

POST http://{server_ip}:{port}/ask
Header: Authorization: Bearer <USER_API_KEY>

Request body (JSON):
  {{
    "prompt": "Your question here",
    "model": "{default_model}",
    "stream": false,
    "parameters": {{
      "temperature": 0.7,
      "max_tokens": 2048,
      "top_p": 0.9
    }}
  }}

Fields model, stream, parameters are optional.
If model is not specified, the first connected model is used by default.

### Example (Python):

```python
import httpx

resp = httpx.post(
    "http://{server_ip}:{port}/ask",
    headers={{"Authorization": "Bearer <USER_API_KEY>"}},
    json={{"prompt": "Hello, how are you?"}},
)
print(resp.json()["response"])
```

### Response (JSON):
  {{
    "response": "Model response text...",
    "model": "{default_model}",
    "usage": {{
      "prompt_tokens": 12,
      "completion_tokens": 345,
      "total_tokens": 357
    }},
    "elapsed_seconds": 4.2
  }}

## 2. Streaming Request (SSE)

Send the same POST /ask with "stream": true.
The response is Server-Sent Events (text/event-stream).

```python
import httpx

with httpx.stream(
    "POST",
    "http://{server_ip}:{port}/ask",
    headers={{"Authorization": "Bearer <USER_API_KEY>"}},
    json={{"prompt": "Tell me a story", "stream": true}},
) as resp:
    for line in resp.iter_lines():
        if line.startswith("data: "):
            print(line[6:])
```

Each chunk:
  data: {{"token": "part of the text", "done": false}}
Last chunk:
  data: {{"token": "", "done": true, "usage": {{...}}, "elapsed_seconds": 4.2}}

## 3. Access the Internet via HTTPS Proxy (OUTGOING MODE)

The server operates as a standard HTTP/HTTPS proxy on port {proxy_port}.
Authentication: username and password (Proxy-Authorization Basic).

Set environment variables:
  HTTPS_PROXY=http://<proxy_user>:<proxy_pass>@{server_ip}:{proxy_port}
  HTTP_PROXY=http://<proxy_user>:<proxy_pass>@{server_ip}:{proxy_port}

Or specify the proxy in code:

```python
import httpx

client = httpx.Client(
    proxy="http://<proxy_user>:<proxy_pass>@{server_ip}:{proxy_port}"
)
resp = client.get("https://api.example.com/data")
print(resp.text)
```

All requests through the proxy use the server's IP (your IP is hidden).

GET http://{server_ip}:{port}/models

---

## 5. Check Server Status

GET http://{server_ip}:{port}/health
(no authorization required)

---

## 6. Error Codes

- 401 — invalid authorization
- 404 — model not connected
- 408 — model response timeout
- 422 — invalid request
- 429 — too many requests
- 502 — model unavailable
- 407 — invalid proxy credentials
- 403 — domain blocked by proxy
